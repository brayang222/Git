Cd = change directory, cambia la posición, también te devuelve al punto inicial
ls = Muestra todas las carpetas y archivos que hay en tu pos
mkdir = crea una carpeta
rmdir = elimina una carpeta
clear = elimina los comandos
flecha arriba = muestra los comandos anteriores
git init = para iniciar repositorio en dicha carpeta
git add . = para añadir todo a el repositorio
git status = para ver el estatus del repositorio
git rm --cached . = para sacar lo que había añadido
git commit -m "agregué..." = agrega un comentario al commit
git log = para ver los datos del commit
code . = para abrir vs Code
git commit --amend = para cambiar el commit

git branch ... = para crear una rama idéntica y hacer modificaciones
git checkout ... = para cambiar de rama
git checkout -b ... = (crea una rama y va directamente a ella)
git branch -m ... = cambiar el nombre de la rama
git branch -d ... = para eliminar una rama

git log -p = para ver todos los cambios (q) para salir
